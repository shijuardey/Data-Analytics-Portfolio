# Donor & Donation Analysis (SQL Capstone Project)

## Project Overview

This project analyzes donor and donation data using PostgreSQL to uncover fundraising trends, evaluate donor behavior, and identify opportunities for improving donor engagement and retention.

The analysis was conducted as a client-focused analytical engagement, with emphasis on data quality assessment, exploratory data analysis (EDA), donor segmentation, and business-driven insights.

## Business Problem

The organization seeks to better understand its donor base and donation patterns in order to:

* Identify high-value donors
* Understand donation behavior across different donor segments
* Evaluate fundraising performance
* Improve donor retention strategies
* Support data-driven fundraising decisions

## Dataset Description

The project uses two related datasets:

### donor_data

Contains donor demographic and profile information:

* ID
* First Name
* Last Name
* Gender
* State
* Job Field
* University
* Donation Frequency
* Second Language

### donation_data

Contains donation transaction information:

* ID
* Donation Amount

The datasets are linked using the **ID** field.

## Key Findings

* High-value donors contribute a significant share of total donations.
* Donation frequency influences overall fundraising performance.
* Certain states generate higher donation volumes than others.
* Missing donor profile information affects segmentation analysis.
* Donor demographics provide valuable fundraising insights.

## Recommendations

1. Strengthen engagement with recurring donors.
2. Focus fundraising efforts on top-performing donor segments.
3. Improve collection of donor profile information.
4. Implement regular data quality monitoring.
5. Develop targeted campaigns based on donor characteristics.
