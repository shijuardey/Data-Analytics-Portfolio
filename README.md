## 1. Donor & Donation Analysis (SQL Capstone Project)

### Project Overview
This project analyzes donor demographics and donation patterns to identify high-value donor segments, evaluate fundraising performance, and provide recommendations for improving donor engagement and retention.

- Null value assessment

- Advanced SQL using Views and CTEs

### Tools Used

- SQL

### Key Insights

- Analyzed donation behavior across demographics.
- Evaluated donation performance by state and frequency.
