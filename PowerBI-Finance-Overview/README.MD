# Finance Overview Dashboard (Power BI)

## Project Overview

This project presents an interactive Power BI dashboard designed to monitor and analyze financial transactions, income, expenses, payment methods, fund classifications, and monthly financial performance.

The dashboard provides stakeholders with a centralized view of financial activities, enabling better decision-making through real-time insights and visual reporting.

## Business Problem

Organizations need a clear understanding of their financial performance to effectively manage resources, control expenses, and identify trends.

This dashboard was developed to answer key business questions such as:

* What is the total income generated?
* What is the total expenditure incurred?
* Which expense categories account for the highest spending?
* Which payment methods are most frequently used?
* How do financial transactions vary over time?
* What are the major contributors to income and expenses?

## Dataset Description

The dataset contains financial transaction records including:

* Transaction Date
* Category
* Amount
* Payment Method
* Fund Class
* Transaction Type (Income/Expense)
